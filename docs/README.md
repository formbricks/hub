# Formbricks Hub Documentation

This documentation is built with [Mintlify](https://mintlify.com).

## Local Development

1. Install the Mintlify CLI globally:
   ```bash
   npm i -g mint
   ```

2. Navigate to the docs directory and start the development server:
   ```bash
   cd docs
   mint dev
   ```

The documentation will be available at `http://localhost:3000`.
